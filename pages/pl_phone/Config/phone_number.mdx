### Phone Number

| Setting | Default | Type | Description |
|---------|---------|------|-------------|
| `PhonePrefix` | "08" | String | prefix number |
| `PhoneLength` | 5 | Number | length of number after prefix |

```lua
Config.PhoneNumber = {
    PhonePrefix = "08", -- prefix
    PhoneLength = 5 -- length of number after prefix
}
```
