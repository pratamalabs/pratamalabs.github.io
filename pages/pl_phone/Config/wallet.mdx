# Wallet Configuration
```lua
Config.Wallet = {
    MAX_TRANSFER      = 10000000,
    MIN_TRANSFER      = 100,
    TX_LIMIT          = 50,
    TRANSFER_COOLDOWN = 10,
}
```

---

## Settings

| Setting | Default | Type | Description |
|---------|---------|------|-------------|
| `MAX_TRANSFER` | `10000000` | Number | Maximum amount per single transfer |
| `MIN_TRANSFER` | `100` | Number | Minimum amount per single transfer |
| `TX_LIMIT` | `50` | Number | Maximum number of transactions stored in history |
| `TRANSFER_COOLDOWN` | `10` | Number | Delay in seconds between each transfer |
