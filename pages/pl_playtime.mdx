## Player Command
- open shop: `playtime`

## Admin Command
- give coins: `givecoins [playerid/citizenid] [amount] `
- remove coins: `removecoins [playerid/citizenid] [amount] `
- set coins: `setcoins [playerid/citizenid] [amount] `

### server cfg
`customize to your needs access admin commands`
``` bash
#Playtime Commands Permission for Specific Players
add_ace identifier.license:xxxxxxxxxxxxxxxx playtimecommand allow
add_ace identifier.steam:110000xxxxxxxxxx playtimecommand allow
```
``` bash
#Playtime Commands Permission for Admin Group
add_ace group.admin playtimecommand allow
```
